# 个人动漫网站

这是一个基于HTML、CSS和JavaScript的个人动漫网站，采用苹果风格的清新设计，适合动漫爱好者展示个人收藏和分享动漫相关内容。

## 项目结构

```
├── index.html          # 首页
├── about.html          # 关于我页面
├── collection.html     # 动漫收藏页面
├── merchandise.html    # 周边商品页面
├── blog.html           # 博客页面
├── contact.html        # 联系页面
├── todo-list.html      # 待办清单页面
├── vercel.json         # Vercel部署配置
└── README.md           # 项目说明
```

## 功能特点

### 1. 首页
- 英雄区域展示个人动漫主题
- 推荐动漫网格展示
- 最近更新区域
- 快速导航菜单

### 2. 关于我
- 个人介绍
- 兴趣爱好展示
- 动漫观看历史时间线

### 3. 动漫收藏
- 动漫卡片展示
- 按类型、年份、评分筛选功能
- 响应式网格布局

### 4. 周边商品
- 手工艺品展示
- 收藏品展示
- 商品详情布局

### 5. 博客
- 博客文章列表
- 侧边栏热门标签和最近文章
- 文章卡片设计

### 6. 联系页面
- 联系信息展示
- 留言表单
- 社交媒体链接

### 7. 待办清单
- 添加新任务
- 标记任务完成/未完成
- 删除任务
- 本地存储持久化

## 技术栈

- **前端框架**: 纯HTML5 + CSS3 + JavaScript
- **样式设计**: 苹果风格，清新简约
- **响应式设计**: 适配手机和电脑端
- **数据存储**: localStorage (待办清单功能)
- **部署平台**: Vercel

## 部署方法

### 1. GitHub 仓库
- 项目已托管在 GitHub: [https://github.com/liangxiao060104-ux/Chaireliang06](https://github.com/liangxiao060104-ux/Chaireliang06)

### 2. Vercel 部署
1. 登录 Vercel 账号 (使用 GitHub 登录)
2. 导入 GitHub 仓库 `liangxiao060104-ux/Chaireliang06`
3. 选择静态网站框架预设
4. 点击 "Deploy" 按钮
5. 部署完成后获得访问链接

## 本地开发

1. 克隆项目到本地
   ```bash
   git clone https://github.com/liangxiao060104-ux/Chaireliang06.git
   ```

2. 打开项目文件夹
3. 直接在浏览器中打开 `index.html` 文件即可查看效果

## 设计风格

- **配色方案**: 淡粉色、薄荷绿、浅蓝色，营造清新的苹果风格
- **布局**: 响应式网格布局，卡片式设计
- **交互**: 平滑过渡动画，悬停效果
- **字体**: 系统默认字体，确保跨平台一致性

## 后续维护

- 向 GitHub 仓库推送新的更改后，Vercel 会自动重新部署
- 可以通过修改 HTML 文件来更新网站内容
- 可以添加新的页面或功能来扩展网站
